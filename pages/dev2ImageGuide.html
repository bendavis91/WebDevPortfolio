    Image Guideline Table

    JPEG IMAGE
    Use a JPEG image when you have many different colors in a picture. Very high definition pictures are best saved as JPEG. When saving pictures its good to name it including the dimensions, for example Sunset_600x338.jpg.
    GIF IMAGE
    Use a GIF format when saving images with few colors or large areas of the same color. In this instance I used an animated gif. Animated GIFS must be under 3MB and max at 540 pixels (although some instances allow more width depending on GIF size). When saving animated gifs you never want the size to be too high or it becomes distorted or blurry.

    PNG IMAGE
    PNG is a good choice for storing line drawings, text, and iconic graphics at a small file size. PNG format is a lossless compression file format, which makes it a common choice for use on the Web. PNG files are commonly used to store graphics for web images. As for all images, it's best to save them and enter a name of the file that represents the size of the image so you know by just reading the file name. For example, I saved the larger image to the right as mario_luigi_L_550x479.png. You don't have to type the L indicating a larger image, but just the dimension sizes would do fine.
